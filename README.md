# 灵可世集 (Lingke Shiji)

基于社区积分的元宇宙本地生活服务平台，连接服务商与消费者。

## 项目简介

灵可世集是灵可生态系统的重要组成部分：
- **善行天下**：记录善行，积累功值
- **灵可币**：基于BSC的代币
- **灵可社区**：封闭邀请制互助平台
- **灵可世集**：元宇宙本地生活服务平台

## 功能特性

- 动态星空粒子背景（Canvas动画）
- 3D标题入场动画
- 展示平台理念：元宇宙本地生活新纪元
- 关键数据：500+积分门槛、1,200+认证服务商、50,000+活跃用户

### 2. 服务商星系

- 温馨民宿（住宿酒店）
- 智慧学堂（教育培训）
- 健康之家诊所（医疗健康）
- 贴心家政（家政服务）
- 数码维修站（数码维修）
- 美丽发艺（美容美发）

### 3. 服务分类星云
8大服务类别：
- 餐饮美食（328家）

### 4. 积分权限体系

| 等级 | 积分要求 | 权限 |
|------|---------|------|
| 普通用户 | 0 | 浏览、消费、评价、收藏 |
| 高级用户 | 100 | 优先推荐、专属优惠、会员活动 |
| 认证服务商 | 500 | 开设店铺、发布服务、获得收益 |

### 5. 特色服务展示

- **虚拟民宿** - 3D全景看房
- **技能NFT** - 认证技能交易

## 技术栈

- **框架**: React + TypeScript + Vite
- **样式**: Tailwind CSS
- **组件库**: shadcn/ui
- **动画**: GSAP (ScrollTrigger)
- **3D效果**: Three.js / React Three Fiber
- **图标**: Lucide React

## 项目结构

```
src/
├── sections/          # 页面区块组件
│   ├── Hero.tsx       # 主视觉区（星空背景）
│   ├── Providers.tsx  # 服务商展示
│   ├── Categories.tsx # 服务分类
│   ├── PointsSystem.tsx # 积分权限体系
│   ├── Features.tsx   # 特色服务
│   ├── CTA.tsx        # 行动号召
│   ├── Footer.tsx     # 页脚
│   └── Navigation.tsx # 导航栏
├── components/        # UI组件
│   └── ui/            # shadcn/ui组件
├── hooks/             # 自定义Hooks
├── lib/               # 工具函数
├── App.tsx            # 主应用组件
├── App.css            # 应用样式
├── index.css          # 全局样式
└── main.tsx           # 入口文件
```

## 安装与运行

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

## 设计特点

- **品牌色彩**：深空黑(#0a0f1a) + 星际金(#c9a87c) + 极光青(#00d4aa) + 星云紫(#8b5cf6)
- **动效系统**：GSAP滚动动画、星空粒子背景、3D变换
- **响应式设计**：完美适配桌面端和移动端
- **交互体验**：悬停光效、平滑滚动、轮播切换

## 与生态系统关联

- 用户需通过 **善行天下** 积累功值
- 使用 **灵可币** 在 **灵可社区** 进行委托互助
- 达到 **500社区积分** 即可在 **灵可世集** 成为服务商
- 任何人都可以注册使用灵可世集进行消费
